# Vaishnavi Annam - Premium Professional Portfolio

A polished, recruiter-focused portfolio for software development internships and entry-level software roles.

## Features
- Premium dark UI with gradients and glassmorphism
- Smooth scroll reveal animations
- Responsive mobile-first layout
- Project showcase section
- Resume download button
- Leadership and achievement highlights

## Customize Before Publishing
1. Replace GitHub, LinkedIn, and portfolio URLs in `index.html`.
2. Add real GitHub and Live Demo links for each project.
3. Update LeetCode count as you solve more problems.
4. Replace the resume PDF in `assets/` whenever you update it.

## Deployment
Upload to GitHub and enable GitHub Pages from the `main` branch, root folder.
